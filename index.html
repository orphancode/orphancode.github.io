Know of scientific code lost in supplementary information, decrepit personal pages or emailed a long time ago in a zip file? That could be useful after all.

Let's preserve scientific code, .

If nothing this could be close to an archaeological experience in modern-times.

Licencing

Obviously, code is subject to licenses. In some (many) cases a licence is not specified in the original code.

I'd recommend contacting the original author regarding this and asking if she agrees on licencing the code with an open source licence (MIT, GPL, etc...).

Contributing

Try adding as much information to the repository's README as possible, always including credit to the original code author.
